MTAutoLabel
===========

Finder service to auto label my movies folder.


Installation
------------

- Clone this repository or download the [workflow][download].
- Copy/Move `AutoLabel.workflow` to ~/Library/Services
- Done :)


Usage
-----

Control-click the folder(s) > Services > AutoLabel
